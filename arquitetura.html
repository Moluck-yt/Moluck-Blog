Organização Estruturada de Computadores

Introdução

No livro do Tanenbaum “Organização Estruturada de Computadores”, ele define um computador digital como uma máquina que pode resolver problemas executando instruções dadas por programadores. É fundamental que qualquer programador compreenda a organização estruturada de computadores para resolver problemas de maneira eficiente.

Programa e Instruções

Um programa é uma lista de instruções para realizar tarefas específicas. Tanenbaum explica que os circuitos eletrônicos de um computador reconhecem e executam diretamente um conjunto limitado de instruções simples. Essas instruções básicas incluem:

  • Somar dois números
  • Verificar se um número é zero
  • Copiar dados de uma parte da memória para outra

Essas instruções formam a linguagem de máquina, que é difícil para humanos aprenderem, mas existem linguagens de baixo nível que facilitam essa compreensão.

Linguagem de Máquina e Projeto de Computadores

Quem projeta novos computadores deve decidir quais instruções incluir na linguagem de máquina, focando em eficiência, desempenho e baixo custo. Segundo Tanenbaum, o que é conveniente para nós não é necessariamente conveniente para as máquinas, levando à criação de seu livro para explicar esses conceitos.

Arquitetura de Computadores

A arquitetura de um computador se refere aos atributos visíveis ao programador, como conjunto de instruções, número de bits usados para representar vários tipos de dados, mecanismos de E/S e técnicas de endereçamento de memória.

Conclusão

Compreender a organização e a arquitetura dos computadores é essencial para desenvolver software eficiente. Tanenbaum fornece uma base sólida para esse conhecimento, permitindo que programadores criem soluções melhores e mais otimizadas.
